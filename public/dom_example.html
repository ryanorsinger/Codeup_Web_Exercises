    setInterval() Run a function over and over every interval

  • var intervalId = setInterval(someFunction, interval);
  • Runs someFunction every "interval" milliseconds, over and over.
  • The setInterval will run until we call clearInterval(intervalId) or the browser closes or crashes.
  • This is used for running a game loop

    setTimeout() Set a delay, and run a function once 

  • var intervalId = setInterval(someFunction, interval)
  • Timeouts run the function once, after the delay.
  • We can interrupt the timeout by using clearTimeout(), just like with intervals.
